# 지능형 기타 톤 코칭 AI 시스템 (Guitar Tone Coach AI)

## 프로젝트 개요
본 프로젝트는 사용자가 동경하는 레퍼런스 기타 음원(목표 톤)과 현재 자신의 기타 톤(연주)을 인공지능이 분석하여, 앰프와 이펙터의 노브(Gain, Bass, Mid, Treble)를 어떻게 조절해야 하는지 시각적이고 직관적인 가이드를 제공하는 동적 웹 UI 기반 코칭 서비스입니다.

## 소스코드 디렉토리 구조
이 레포지토리는 시스템 구동에 필요한 핵심 소스코드와 1-Click 도커(Docker) 배포 환경만으로 깔끔하게 구성되어 있습니다. (방대한 훈련용 데이터셋은 용량 최적화를 위해 제외되었습니다.)

- `app.py` : Streamlit 기반 웹 애플리케이션의 메인 진입점. (UI 렌더링, 오디오 플레이어, 코칭 가이드 시각화 로직)
- `src/` : 보이지 않는 백엔드에서 동작하는 AI 코어 모듈 폴더. (HTDemucs 기반 음원 분리, 멜-스펙트로그램 전처리, 멀티태스크 추론 로직 등 포함)
- `best_multi_task_resnet_model.pth` : 5천 개의 하이브리드 데이터로 미세 조정(Fine-Tuning)된 ResNet-18 기반의 핵심 AI 가중치 모델.
- `Dockerfile` & `docker-compose.yml` : 파이썬 버전 및 OS 환경 종속성 충돌을 완벽히 방지하기 위한 컨테이너 환경 격리 설정 파일.
- `requirements.txt` : 구동에 필요한 파이썬 라이브러리 명세서.
- `run_app_docker.bat` : 도커 명령어를 몰라도 마우스 더블클릭 한 번으로 전체 시스템을 구동시켜주는 윈도우용 편의 스크립트.

## 주요 기술 스택
- **AI / 오디오 처리:** PyTorch, ResNet-18, HTDemucs(6-Stem), Librosa, torchaudio
- **웹 UI / 시각화:** Streamlit, Matplotlib
- **배포 / 인프라:** Docker, Docker-compose

## 빠른 실행 방법 (Quick Start)
복잡한 파이썬 환경 설정이나 라이브러리 설치가 필요 없습니다. 도커(Docker)를 이용해 즉시 구동할 수 있습니다.

1. Docker Desktop 프로그램이 실행되어 있는지 확인합니다.
2. 다운로드받은 소스코드 폴더 내에 있는 `run_app_docker.bat` 파일을 더블클릭합니다.
3. 터미널 창이 뜨고 서버가 성공적으로 켜졌다는 메시지가 나오면, 웹 브라우저를 열고 `http://localhost:8501/` 에 접속합니다.
4. 오디오 파일을 업로드하여 AI 기타 톤 코칭 서비스를 직접 체험해 보세요.

## 주요 기능 포인트
- **1-Click 환경 격리:** `.dockerignore`를 통해 최적화된 컨텍스트 빌드로 수 분 이상 걸리는 빌드 타임을 1~2초 수준으로 단축.
- **명시적 코칭 가이드:** 목표 톤과 현재 톤의 오차가 7.5% 이상 발생하는 파라미터에 대해 "Gain을 10% 더 올리세요"와 같은 직관적인 Actionable Feedback 제공.
- **직관적 비교 차트:** Gain, Bass, Mid, Treble 4종의 설정값을 막대그래프(Bar Chart)로 비교 시각화하여 음향적 차이를 한눈에 파악.
